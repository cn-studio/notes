Html

網頁設定
<!DOCTYPE html> (html5版本)
<html>
      <head>
      <meta http-equiv="Content-Type" content="text/html"; charset=UTF-8"> (萬國編碼)
      <title>網站標題</title> (網站標題)
      <meta name="dcscription" content="敘述"> (網站描述文字)
      <meta name="keywords" content="關鍵字1,關鍵字2,關鍵字3"> (網站描述文字)
      <meta property="og:title" content="FB的標題"> (FB相關)
      <meta property="og:description" content="FB的描述"> (FB相關)
      <meta property="og:typel" content="website"> (FB相關)
      <meta property="og:url" content="FB上的網址"> (FB相關)
      <meta property="og:image" content="分享至FB所呈現的圖片"> (FB相關)
      <meta http-equiv="X-UA-Compatible" content="IE=Eege"> (將使用者的ie變更為最新)
      <meta name="viewport" content="width=device-width, initial-scale=1"> (RWD設定) (瀏覽器顯示寬度=裝置的寬度)，(縮放比例為1)
      <link rel="stylesheet" href="mystyle.css"> (外部連結CSS)
      <link rel="shortcut icon" href="image/title_icon.ico"> (網頁名稱前圖示)
      </head>
      <body>
      </body>
</html>

</dl> 描述型情單 dl為容器最外層，dt為左欄的敘述，dd為右欄的敘述

<header></header> (可用於網頁標題)

<nav></nav> (選單列表)

<main></main> (主要內容)

<section></section> (內容區)

<article></article> (文章)

<footer></footer> (頁尾)

超連結

<a></a> (可用於網頁標題)

<a href="目標" target="_blank" title="連結顯示文字"></a> (超連結)

<a href="#test"></a> (連結至該檔案id為test的元件)

target="_blank" 設定此屬性時，點擊後會開啟新視窗

<audio src="來源" controls autoplay loop> (音源)

<video src="來源"> (影像)

自定義參數

<p> data-test="1"

data-後方的參數名稱，與參數內的數值可自定義
表單元件
<form action="導入的資料庫位置">
      <label for="mail">電子郵件</label> (for內的數值與input的name呼應，可提高使用者體驗)
      <input type="text" name="mail"> (輸入的種類是文字，資料庫的名稱為mail)
      <input type="tel" > (輸入的種類是數字)
      <input type="mail" > (輸入的種類是信箱)
      <input type="submit" value="下一步"> (輸入的種類是按鈕 value="按鈕上的文字描述")
      <input type="radio" name="sex" value="man"> (單選元件)
      <input type="radio" name="sex" value="women"> (單選元件)       單選方塊，同一欄元件當參數name相同時，只能選擇一個元件
      <input type="checkbox" name="movie" value="hobby"> (複選元件)
      <input type="radio" name="music" value="hobby"> (複選元件)       複選方塊，同一欄元件下可勾選多格條件
      <select name="birth">
            <option value="1991">1991</option>
            <option value="1992">1992</option>
      </select>       此為多選型下拉式選單，value為後端應用參數
      <textarea name="content" cols="30" row="10"> (多行文字輸入區，cols=單行文字量，row="行數")
      <button></button> (按鈕元件)
      <input placeholder="請輸入電子郵件">(placeholder可輸入提示的內容)
      <input readonly>(僅能閱讀不能填寫)
      <input disabled>(無法選取)
      <form>
            <fieldest disabled></fieldest>(全表單禁用)
      </form>
