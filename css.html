overflow | 卷軸

overflow:hidden (隱藏超出範圍的內容，不產生卷軸)

overflow:auto (超出範圍的x、y軸會產生選軸)

overflow:visable (超出範圍不隱藏，不產生卷軸)

overflow:scroll (超出範圍內容會產生卷軸)

text | 文字排版

text-indent:'n'px (首行縮排'n'px)

letter-spacing: 'n'px (文字間距'n'px)

line-height: 'n'px (文字行距'n'px)

text-overflow: clipe (超出的範圍直接切斷)

text-overflow: ellipsis (超出的範圍改為…)

.ellipsis {
      overflow:hidden;
      white-space: nowrap;
      text-overflow: ellipsis;
} 一併設定 white-space: nowrap; 及 text-overflow: ellipsis; 才能呈現此效果

writing-mode: horizontal-tb (預設樣式)

writing-mode: vertical-rl (由右至左垂直書寫)

writing-mode: vertical-lr (由左至右垂直書寫)

文字換行

word-break: break-all; (超出範圍的"字元"換行)

word-wrap:break-word; (超出範圍的"單字"換行)

white-space:pre; (保留原始文件的空白與換行，功能與Html中的<pre>相同)

white-space:nowrap; (強迫文字不換行)

white-space:pre-wrap; (保留所有空白，自動換行)

white-space: pre-line; (將多個空白轉換為1個空白)

font | 文字字體

font-style:normal (字體樣式，預設為normal，斜體為italic)

background-image:url(路徑) (背景圖片)

background-size: contain (背景完整呈現)

background-repeat: repeat(背景重複)

background-repeat: repeat-x (背景x軸重複)

background-repeat: repeat-y (背景y軸重複)

background-attachment: scroll (背景捲動-預設屬性)

background-attachment: fixed (背景捲動-固定不捲動)

cursor | 游標樣式

cursor: pointer (游標轉換為可點選的樣貌)

cursor: not-allow (無法點擊)

cursor: none (無游標)

display | 排版

display: none (物件不顯示，也不占版面)

display: inline (項目橫式排列)

disaply: inline-block (項目橫式排列，且被賦予空間)

disaply: block (項目直式排列，且被賦予空間)

float | 浮動排版

float: none (不浮動)

float: left (向左浮動)

float: right (向右浮動)

clear: left (清除向左浮動)

clear: right (清除向右浮動)

clear: both (清除左右浮動)

shadow | 容器陰影

box-shadow: x軸距離 y軸距離 模糊 顏色 inset inset 為內陰影屬性

text-shadow: x軸距離 y軸距離 模糊 顏色

transform | 動態效果

transform: translate(n, n) (x,y軸的移動量)

transform: scale(n, n) (x,y軸的大小，參數為0~1)

transform: rotate('n'deg) (選轉的角度)

transform: skew('n'deg, 'n'deg) (x,y軸傾斜的角度)

transform: rotate('n'deg) scale(n, n) skew(n, n) (多重屬性)

transition | 動態時間軸

transition: height 3s 2s, width 3s 2s (高度變化3秒、延遲2秒，寬度變化3秒、延遲2秒)

transition-property: all (全部有變動的屬性進行變化)

transition-duration: 2s (持續'2'秒)

transition-delay: 2s (延遲'2'秒執行)

transform: rotate('n'deg) scale(n, n) skew(n, n) (多重屬性)

transition-timing-function: ease (動畫播放速率)

動畫速率-內建屬性
ease (0.25, 0.1, 0.25, 1.0)
linear (0.0, 0.0, 1.0, 1.0)
ease-in (0.42, 0, 1.0, 1.0)
ease-out (0, 0, 0.58, 1.0)
ease-in-out (0.42, 0, 0.58, 1.0) 此參數可動態呈現，屬性的數值變化
animation | 動畫參數

@keyframes test (建立一個名為test的動畫參數)

@keyframes test{
      0%{width: 0%}
      25%{width: 25%}
      50%{width: 50%}
      75%{width: 75%}
      100%{width: 100%}
}
animation | 動畫代入

animation-name: test (帶入動畫名稱為test的動畫參數)

animation-duration: 3s (持續3秒)

animation-delay: 2s (延遲2秒)

animation-iteration-count: infinte (動畫重複次數，infinte為重複播放)

animation test 3s 2s linear infinte (帶入動畫名稱為test的動畫，持續3秒，延遲兩秒，加速度播放，連續播放)

animation-direction:normal (動畫播放方向 normal:預設模式 reverse:反轉，由100%~0%
alternate:正反轉輪流播放，奇數次為 0% 到 100%，偶數次為 100% 到 0%，若動畫播放次數只有一次就只會正常播放
alternate-reverse:alternate 的相反，奇數次為 100% 到 0%，偶數次為 0% 到 100%，若動畫播放次數只有一次就只會反轉播放)

animation-fill-mode:none (動畫執行後的狀態，預設none:回到初始狀態、forwards:停留在動畫執行完畢的狀態)

z-index | 圖層順序

z-index: 1 (數字越大，圖層越上層)

box-sizing | 盒模型

box-sizing: border-box (盒模型本體包含padding 與 border)

Flex | 排版系統
啟用此排版系統，需將父層設定display:flex row為水平排版

flex-direction: row (由左至右排版-預設值)

flex-direction: row-reverse (由右至左排版-位向反轉)

column為垂直排版

flex-direction: column (由上至下排版)

flex-direction: column-reverse (由下至上排版-位向反轉)

justify-content 子項目的水平排序，當flex-direction:row 才起作用

justify-content: flex-start (由flex-direction定義的起點對齊)

justify-content: flex-end (由flex-direction定義的終點對齊)

justify-content: center (水平對齊)

justify-content: space-around (內外均且留白)

align-content 子項目的垂直排序，當flex-direction:culumn 才能起作用

align-items: stretch (垂直撐滿父層高度)

align-items: flex-start (由flex-direction定義的起點對齊)

align-items: flex-end (由flex-direction定義的終點對齊)

align-content: flex-start (由flex-direction定義的起點對齊)

align-content: flex-end (由flex-direction定義的終點對齊)

align-content: center (垂直置中)

flex-wrap內容物過多的處理方式

flex-warp: no-wrap (內容物不換行-預設)

flex-wrap: wrap (內容物容量過大會自動換行)

flex-warp: warp-reverse (自動換行外，位向反轉)

Flex | 子項目元件
flex-grow 內容物的是否自動填滿空格

flex-grow: 0 (不成長-預設)

flex-grow: 1 (會自動成長)

flex-shrink 當內容物撐爆容器後，是否進行縮小

flex-shrink: 0 (不縮小)

flex-shrink: 1 (縮小-預設)

flex-basis 單一區塊寬度

flex-basis:500px (寬度500px)

flex-basis:50% (寬度50%)

flex-basis:auto (依照內容寬度呈現-預設值)

order 元件排序

order:1 (數字越小，排序越前面)

margin 此排版下的間距

margin-right:auto (抓出多餘空間補在右方)

margin:auto (抓出多餘空間補在上下左右)

margin:auto 0 (抓出多餘空間補在上下，左右無間距)

align-self 物件的對齊方式

align-sleft: stretch (垂直撐滿父項目高度)

align-sleft: flex-start (由flex-direction定義的起點對齊)

align-sleft: flex-end (由flex-direction定義的起點對齊)

align-sleft: center (垂直置中)
