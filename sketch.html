Sketch

快捷鍵

command + G(解散群組)

按住command + 物件(可直接選擇群組內的物件)

shift + command + l，(或者選擇旁邊圖層按著option按鍵會出現鎖頭，點選後就能鎖定)

option + command + 方向鍵 上 或 下對圖層進行上下移動

control + option + command + 方向鍵上或下(將圖層移動到最上或最下層)

option + command + c(複製圖層樣式)

option + command + v(貼上圖層樣式)

control + c(顏色滴管)

點選文字按右鍵“convert to outlines”(文字建立外框)
點選線段按layer，並選“convert to outlines”(線段建立外框)
將兩個元件選取，從上方工具區或選擇右鍵點選“mask”(建立遮色片，建立後會成為一個群組)
