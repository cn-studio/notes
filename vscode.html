Live Sass Compiler | 安裝流程

1.安裝 Live Sass Compiler
2.接著進入 Visual Studio Code的喜好設定->設定 - Mac用戶
2.可從左上角的「檔案」選單內找到「喜好設定」選項 - Windows用戶
3.點選 setting.json進入設定介面
4.貼上此段程式碼
{
"liveSassCompile.settings.formats":[
{
"savePath": "/css"
}
]
}

快捷鍵(windows)

ctrl按鍵 + p(搜尋檔案)

ctrl按鍵 + enter按鍵(強制插入空白的一行內容)

ctrl按鍵 + shift按鍵 + f(所有專案的搜尋）

alt按鍵 + shift按鍵 + f(快速調整程式碼排版)

alt按鍵 + shift按鍵 + l(快速啟動瀏覽器，需安裝Preview on Web server插件)

ctrl + d(選取相同字串元素)

ctrl + l(選取整行程式碼)

alt按鍵 + 上 或 下鍵(選取的區域上下移動)

ctrl按鍵 + `(開啟問題、輸出、偵錯主控台、終端機等功能)

快捷鍵(mac)

command + p(搜尋檔案)

command + enter(強制插入空白的一行內容)

option按鍵 + shift按鍵 + f(快速調整程式碼排版)

command + shift + l(快速啟動瀏覽器，需安裝Preview on Ｗeb server插件)

command + d(選取相同的字串元素)

command + l(選取整行程式碼)

option + 上 或 下鍵(將選取的區塊上下移動)

command + `(開啟問題、輸出、偵錯主控台、終端機等功能)
