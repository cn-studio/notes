SCSS

環境設定

因SASS/SCSS需經過編譯處理，詳見 設定方法

註解撰寫
.header{
      height:100px
            ul{
                  height:200px;
            }
} class="header"的高度100px,.header ul的高200px
變數撰寫

支援+ - *等數學符號

$maincolor:rgb(245, 245, 245)($名稱 : 屬性)

.header{
      color:$maincolor;
} 可直接將$maincolor帶入屬性
檔案拆解

@import "檔案名稱"可在scss中帶入另一隻scss的檔案

@improt 'reset';
工具 | mixin

@mixin 自訂名稱($變數名稱, $變數名稱)(可自行定義的快速工具包)

@mixin test($height, $width){
      height: $height;
      width: $width;
      margin: 0;
} 需要用@include參數進行調用
工具調用 | include

@include 名稱(名稱為 mixin 所定義的參數)

ul {
      color: $maincolor;
      @include test(100vh, 100vw);
} ul此標籤的高 = 100vh, 寬 = 100vw 需與@mixin 進行搭配
載具 | content

@content 帶入客製化內容

開設一個RWD斷點檔案(_screen.scss) @mixin pad{
      @media (max-width: 768px){
      @content
} 回到主要的css樣式中 @import ‘screen’; (並帶入此檔案)
body{
      @include pad(){
            background-color: rgb(10, 10, 10);
}; 在@mixin pad所定義的寬度下，背景會改變顏色);
偽元素
ul{
      &:after{
            .......
      }
}

&:after{...} (相同於 css中的::after)
