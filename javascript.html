if (判斷式)

      要執行的事件}
      else{
      要執行的事件
} 單一判斷式可單用if，多個判斷式可另外加else if

switch (判斷式)

switch(參數){
      case '參數':
            要執行的事件;
            break;
      default:
            要執行的事件;
            break;
} 當條件是多項可預期的參數，switch執行效果較好 default同else，當前條件都沒有滿足時，執行default內的事件
